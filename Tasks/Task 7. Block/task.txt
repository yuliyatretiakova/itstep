В папке img лежит 4 картинки с различным расположением блоков (HTML менять нельзя!!!).
Выполнить следующие задания:
Каждую картинку нужно получить следующими способами:
1. Класс inner - position: absolute, в классах inner1..inner4 можно использовать только CSS-свойства top и left
2. Класс inner - position: absolute, в классах inner1..inner4 можно использовать только CSS-свойства margin
3. Класс inner - position: relative, в классах inner1..inner4 можно использовать только CSS-свойства top и left
4. Класс inner - position: relative, в классах inner1..inner4 можно использовать только CSS-свойства margin

Итого у Вас должно получиться 4 картинки по 4 способа = 16 вариантов
